# My Obsidian Vault

This is a synchronized Obsidian vault managed with GitHub Actions.

## Structure

- **Engineering/**: Engineering team documentation including onboarding and deployment guides
- **Operations/**: Operations policies and procedures
- **.obsidian/**: Obsidian application configuration
- **.github/workflows/**: GitHub Actions automation for sync

## Getting Started

1. Clone this repository
2. Open the vault in Obsidian
3. Check out the various documentation sections

## Automation

This vault is automatically synchronized using GitHub Actions. See `.github/workflows/kuzana-sync.yml` for details.
