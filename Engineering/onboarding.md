# Engineering Onboarding Guide
Welcome to the Engineering team! This guide will help you set up your local development environment.
## 1. Local Setup
Ensure you have the following prerequisites installed:
* Git (v2.30+)
* Docker Desktop
* Node.js (v18+)
## 2. Setting Up Variables
Ask your team lead for the secret keys and populate your `.env` file accordingly. Keep the `.env` private and never commit it to the repository.
